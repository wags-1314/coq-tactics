Tactics Cheatsheet

intros

intros moves hypotheses/variables from goal to context. 
(* begin details *)

forall P : Type, P -> P


forall P : Type, P -> P

  
P: Type
H: P
P
First intros application removes the universal quantifier. The next application moves the first part of the conditional as a hypothesis. 
  apply H.
Qed.
(* end details *)

reflexivity

reflexivity finishes a proof if a equality is reached i.e. x = x. 
forall n : nat, n = n


forall n : nat, n = n

  
n: nat
n = n

  reflexivity.
reflexivity finishes the proof now that the goal is of form x = x 
Qed.

simpl

simpl simplifies computations in a goal 
Nat.add = 
fix add (n m : nat) {struct n} : nat :=
  match n with
  | 0 => m
  | S p => S (add p m)
  end
     : nat -> nat -> nat

Arguments Nat.add (_ _)%nat_scope



forall n : nat, 0 + n = n


forall n : nat, 0 + n = n

  
n: nat
0 + n = n

  
n: nat
n = n
By the definition of addition in natural numbers, 0 + n can be simplified to n. simpl just simplifies 0 + n. 
  reflexivity.
Qed.
simpl can not simplify further an expression that can not be computed from the definition of the function in the expression 
forall n : nat, n + 0 = n


forall n : nat, n + 0 = n

  
n: nat
n + 0 = n

  
n: nat
n + 0 = n
n + 0 in the goal remains n + 0, because it can not be simplified further by the definition of addition of natural numbers 
Abort.
simpl in H applies simpl in the hypothesis H 
forall n : nat, 0 + n = n -> n = n


forall n : nat, 0 + n = n -> n = n

  
n: nat
H: 0 + n = n
n = n

  
n: nat
H: n = n
n = n

  reflexivity.
Qed.

rewrite

rewrite H uses an equality H: x = y to rewrite the goal i.e all occurences of x are replaced with y. rewrite H1 in H2 does the same but the hypothesis H2 is rewritten instead. ‚Üí or <- can be used to specify the direction of rewriting 
forall m n o : nat, m = n -> n = o -> m = o


forall m n o : nat, m = n -> n = o -> m = o

  
m, n, o: nat
m = n -> n = o -> m = o

  
m, n, o: nat
H1: m = n
H2: n = o
m = o

  
m, n, o: nat
H1: m = n
H2: m = o
m = o
replaces m with n in H2 
  
m, n, o: nat
H1: m = n
H2: m = o
o = o
replaces m with o in the goal 
  reflexivity.
Qed.

destruct

destruct term as ... eqn:H splits an inductively defined term into different subgoals. Analogous to proof by cases. eqn:H saves H as the current case 
forall b : bool, (b && true)%bool = b

b: bool
(b && true)%bool = b

  
b: bool
H: b = true
(true && true)%bool = true
b: bool
H: b = false
(false && true)%bool = false

  
b: bool
H: b = true
(true && true)%bool = true

    
b: bool
H: b = true
true = true
 reflexivity.
  
b: bool
H: b = false
(false && true)%bool = false

    
b: bool
H: b = false
false = false
 reflexivity.
Qed.

induction allows us to prove by induction on an inductively defined type

  (* [induction] generates two subgoals:
     - Case 1: n = 0

               and a goal whith S(n') *)
  
0 + 0 = 0
n': nat
IHn': n' + 0 = n'

unfold

unfold opens up a definition in the goal or in a hypothesis. It is different from simpl because unfold midlessly opens up a definition like a macro, whil e simpl tries to simplify the expression 
Definition square n : nat := n * n.



square 5 = 25

5 * 5 = 25

  
25 = 25
 reflexivity.
Qed.

apply

apply applies a hypothesis or lemma on the goal
If the hypothesis/lemma is of the form H: A -> B and the goal is B, apply H would will replace B with A, because as we know A implies B it is sufficient to prove A in order to prove B. 
forall A B : Prop, (A -> B) -> A -> B


forall A B : Prop, (A -> B) -> A -> B

  
A, B: Prop
H1: A -> B
H2: A
B

  
A, B: Prop
H1: A -> B
H2: A
A

  apply H2. (* H2 is A, and because the goal is already a hypothesis, we can stop
               the proof here *)
Qed.
apply H1 in H2 applies the hypothesis/lemma H1 on the hypothesis H2. If H1 is of the form A -> B and H2 is A, then H2 is replaced with B, because we know A -> B and we know A, therefore we can infer B 
forall A B : Prop, (A -> B) -> A -> B


forall A B : Prop, (A -> B) -> A -> B

  
A, B: Prop
H1: A -> B
H2: A
B

  
A, B: Prop
H1: A -> B
H2: B
B

  apply H2.
Qed.

symmetry

symmetry changes x = y to y = x. It is used in conjunction with apply because apply cannot match x = y with y = x 
forall n m : nat, n = m -> m = n


forall n m : nat, n = m -> m = n

  
n, m: nat
H1: n = m
m = n

  (* apply H1 would fail here *)
  
n, m: nat
H1: n = m
n = m

  apply H1.
Qed.

injection

injection H reasons on the equalities of values of inductively defined types. If H is S n = S m, then injection H would generate a new hypothesis n = m 
Inductive Silly : Type :=
  Funny (n m: nat).


forall a b c d : nat, Funny a b = Funny c d -> a = c


forall a b c d : nat, Funny a b = Funny c d -> a = c

  
a, b, c, d: nat
H1: Funny a b = Funny c d
a = c

  
a, b, c, d: nat
H2: a = c
H3: b = d
a = c
 (* [injection] generates two new hypothesis [a = c] and
                            [b = c] *)
  apply H2.
Qed.

discriminate

discriminate H will prove any goal if the context contains a trivially false equality 
forall n : nat, 0 = S n -> 2 + 2 = 5


forall n : nat, 0 = S n -> 2 + 2 = 5

  
n: nat
H: 0 = S n
2 + 2 = 5

  discriminate. (* 0 can never equal S n *)
Qed.
